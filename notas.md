# Notas das Aulas

## Aula 9.1 — Git Básico na Prática

- git init: inicializa um repositório local
- git add: move o arquivo para a área de staging
- git commit: salva um snapshot no histórico
- git push: envia as alterações para o repositório remoto
- git status: mostra o estado atual dos arquivos
- git log: exibe o histórico de commits


## Tabela de Comandos

| Comando           | O que faz                        |
|-------------------|----------------------------------|
| git init          | Cria repositório local           |
| git add           | Adiciona arquivo ao staging      |
| git commit        | Registra alterações no histórico |
| git push          | Envia para o repositório remoto  |
| git status        | Mostra status dos arquivos       |
